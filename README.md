# 🧠 Generative AI Tools – Data Analysis

This project analyzes a dataset of Generative AI tools to identify patterns in categories, modalities, and API availability.  
The objective is to extract structured insights that reflect trends in the evolving AI ecosystem.

---

## 📌 Problem Statement

With the rapid growth of Generative AI tools, it becomes difficult to understand:
- Which categories dominate the market  
- How different modalities (text, image, video, etc.) are distributed  
- The accessibility of tools through APIs or open-source availability  

This project addresses these questions through exploratory data analysis and visualization.

---

## 📊 Approach

- Data cleaning and preprocessing using **Pandas**  
- Exploratory Data Analysis (EDA)  
- Visualization using **Matplotlib** and **Seaborn**  
- Comparative analysis across categories and modalities  

---

## 📈 Key Insights

- Text-based tools show the highest concentration, indicating current market focus  
- Multimodal tools are emerging but remain less prevalent  
- High API availability reflects strong emphasis on integration and scalability  
- Open-source tools are limited but contribute to innovation and accessibility
